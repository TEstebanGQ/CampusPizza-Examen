## Descripcion
Sistema pa gestionar pizzeria, productos, pedidos y clientes.

## Problema
No hay sistema pa controlar inventario ni pedidos. Todo manual y desorganizado.

## Solucion
Base de datos que guarda todo: pizzas, panzarottis, bebidas, combos, pedidos, clientes.

## Caracteristicas
- Registra productos y ingredientes
- Maneja adiciones (extra queso, salsas)
- Crea combos con precio especial
- Guarda pedidos pa comer ahi o recoger
- Control del menu
